Avantages des micros-services

  • Un micro-service possède un rôle
  • Faible nombre de lignes
  • S'utilise très facilement dans un conteneur
  • Brique de communication standard (généralement HTTP)
  • Coût minimal de réécriture
  • scalabilité facilité

Démonstration du cloud pour Dev:
Clever-cloud

Clever-cloud

  • Plateform As A Service
  • startup française
  • il suffit de faire "git push" et l'appli run
  • Nombreux languages supportés
  • 2 localisations: Canada et France

Faire tourner une application revient à:

$ git remote add clever git+ssh://git@push.clever-cloud.com/app_4e2ab.git
$ git push clever master
